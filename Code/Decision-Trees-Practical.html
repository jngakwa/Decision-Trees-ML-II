Cleaning the dataset

Filter out the required variables to help find insights to the question and ensure the variables are appropriate for modeling.

mini_df <- df %>%
  mutate(
    
    # Y-variables
    ALCFLAG = factor(ALCFLAG), 
    MRJYDAYS = factor(MRJYDAYS),
    
    # Regression target cleanup
    IRCIGFM = ifelse(IRCIGFM <= 30, as.numeric(IRCIGFM),
                ifelse(IRCIGFM %in% c(91, 93), 0, NA)),
    
    # convert demographics to factors
    IRSEX = factor(IRSEX),
    NEWRACE2 = factor(NEWRACE2),
    INCOME = factor(INCOME),
    POVERTY3 = factor(POVERTY3),
    EDUSCHGRD2 = factor(EDUSCHGRD2),
    EDUSCHLGO = factor(EDUSCHLGO),
    IMOTHER = factor(IMOTHER),
    IFATHER = factor(IFATHER),
    GOVTPROG = factor(GOVTPROG),
    PDEN10 = factor(PDEN10),
    COUTYP4 = factor(COUTYP4),
    
    # Numeric
    EDUSKPCOM = as.numeric(EDUSKPCOM)
  ) %>%
  drop_na()

The classification tree identifies school grade level (EDUSCHGRD2) as the only variable used to predict alcohol use, highlighting its importance among demographic factors. The model splits students into three groups: younger students in grades 1–3, intermediate students in grades 4–5, and older students in grades 6–10. The probability of alcohol use increases steadily across these groups, with the youngest students showing very low usage rates (about 6%), middle grades showing moderate usage (about 17%), and older students showing the highest likelihood (about 34%). Although the model captures a clear trend between age and alcohol use, the overall misclassification error rate of about 24.9% indicates that demographic variables alone provide only moderate predictive power.

Prune the tree then evaluate the test data on the pruned tree

cv_tree <- cv.tree(dem_tree, FUN = prune.misclass)

best_size <- cv_tree$size[which.min(cv_tree$dev)]

pruned_tree <- prune.misclass(dem_tree, best = best_size)

The confusion matrix shows that the model predicts only the majority class (no alcohol use) for all observations. This results in an accuracy of approximately 75%, but the model fails to identify any instances of alcohol use. This behavior reflects class imbalance in the dataset, where most youth report not using alcohol, as well as the limited predictive power of demographic variables alone.

Bagging

bag_model <- randomForest(
  ALCFLAG ~ IRSEX + NEWRACE2 + INCOME + POVERTY3 +
    EDUSCHGRD2 + EDUSCHLGO + EDUSKPCOM +
    IMOTHER + IFATHER + GOVTPROG +
    PDEN10 + COUTYP4,
  
  data = train,
  ntree = 300,
  mtry = 12,   
  importance = TRUE

While it’s more complex than the single tree, it performs worse than a blind guess. The accuracy is approximately 71%, which is lower than the 75% baseline. Looking at the confusion matrix, the model is struggling to identify drinkers as it missed roughly 80% of the actual drinkers in the test set.

## rf_pred    0    1
##       0 1189  366
##       1   52   43

The random forest model shows limited ability to identify alcohol users. While it correctly classified 76.9% of non-alcohol users, it only accurately identified 12.2% of actual alcohol users, misclassifying the remaining 87.8% as non-users.

varImpPlot(dem_rf)

These plots identify which features most influence the random forest model’s decisions. EDUSCHGRD2, which represents the respondent’s level of education, is the top predictor in both charts by a considerable margin, meaning it has the strongest impact on predicting alcohol use. Variables like POVERTY3, COUTYP4, and INCOME provide solid secondary value, whereas features at the bottom such as IMOTHER and IRSEX have very little influence on the outcome.
